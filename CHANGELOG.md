# Changelog

## v2.0.0

### Added
- Share capture functionality
- Extensible Capture adapters
- Instagram specialized support
- Flashcards creation and management
- Due-card Review queue
- Favorites and Archive systems
- Settings screen with capture preferences
- Incomplete Captures surfacing on Home

### Changed
- Silent capture is now the default
- Optional capture prompt for incomplete captures
- Refined Memory detail editor
- Improved Home screen layout with sections

### Fixed
- Duplicate saves and captures via state guards
- Navigation race conditions
- Room threading and Coroutine scope issues
- UI state persistence during configuration changes
